      Optimización del juego

      Afina la máquina virtual de Java (recolector G1 estilo Aikar) para menos tirones y mejor rendimiento. Máxima reserva toda la RAM al arrancar: más fluidez a cambio de un arranque algo más lento.

      Prioridad alta del juego

      Ejecuta el juego con prioridad de CPU por encima de lo normal.

      Bloquear servicios de Mojang

      Impide que el juego contacte autenticación, sesión y telemetría. Desactívalo solo si necesitas skins online.

      Importar tu versión

      Selecciona el .jar de tu copia y dile al launcher qué versión es para descargar sus librerías y recursos (desde los servidores oficiales, sin cuenta).

      Descargar versión oficial

      Se descarga completa (jar, librerías y recursos) desde los servidores oficiales de Mojang y queda lista para jugar, incluso sin conexión.
